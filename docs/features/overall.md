Feature 1: Core To-Do List Management

Requirements

As a user, I should be able to create, edit, and delete tasks.

As a user, I should be able to mark tasks as complete.

As a user, I should be able to view all tasks in a list.

Feature 2: Sub-Tasks + Progress Tracking

Requirements

As a user, I should be able to add sub-tasks to a task.

As a user, I should be able to mark sub-tasks as complete.

As a user, I should be able to see a progress bar for a task based on completed sub-tasks.

Feature 3: “Currently Working On” / Kanban Column

Requirements

As a user, I should be able to mark a task as “In Progress.”

As a user, I should be able to move tasks between To-Do, In Progress, and Completed columns.

As a user, I should be able to see my current focus task at a glance.

Feature 4: Calendar View (Read-Only)

Requirements

As a user, I should be able to view tasks on a calendar based on due dates.

As a user, I should be able to click a calendar task to see its details.
